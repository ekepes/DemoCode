add()
Adds an object to the collection

clear()
Removes all objects from the collection

contains()
Returns true if a specified object is an element within the collection

isEmpty()
Returns true if the collection has no elements

iterator()
Returns an Iterator object for the collection which may be used to retrieve an object

remove()
Removes a specified object from the collection

size()
Returns the number of elements in the collection

Notes:
* Sets are not ordered
* Sets do not have multiple instances of the same value (i.e. a value can only be added once)
* Adding a value a second time is ignored, not an error.
